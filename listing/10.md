

```
Что выведет данная программа и почему?
func update(p *int) {
  b := 2
  p = &b
}

func main() {
  var (
     a = 1
     p = &a
  )
  fmt.Println(*p)
  update(p)
  fmt.Println(*p)
}

```

Ответ:
```
Выведет 1 1 потому что передали поинтер и изменили не значение на которое он указывает а то куда он указывает после завершения функции это пропадет
```
