

```
Чем отличаются буферизированные и не буферизированные каналы?

```

Ответ:
```
По умолчанию каналы не буферизованы, это означает, что они будут принимать отправления ( chan 
<- ), только если есть соответствующий прием ( <- chan ), готовый принять отправленное значение. Буферизованные каналы принимают ограниченное количество значений без соответствующего приемника для этих значений.

```
