Что выведет данная программа и почему?


func someAction(v []int8, b int8) {
  v[0] = 100
  v = append(v, b)
}

func main() {
  var a = []int8{1, 2, 3, 4, 5}
  someAction(a, 6)
  fmt.Println(a)
}


```

Ответ:
```
выведет 100 2 3 4 5
потому что слайс передается в функцию по значению но ссылка на массив остается та же а значит внутри функции он изменится но когда происходит аппенд создается новый массив и новый слайс локальный
